# 人生设计蓝图 H5

一个单文件静态 H5，用斯坦福人生设计课的对话节奏，帮助用户完成：

- 四个仪表盘：健康、工作、娱乐、爱
- 真问题雷达：区分重力问题和可设计问题
- 工作观与人生观指南针
- 心流与能量地图
- 三个五年奥德赛计划
- 可复制、可下载的个人人生设计蓝图

## 本地使用

直接用浏览器打开 `index.html` 即可。

如果想用本地服务预览：

```bash
python3 -m http.server 8024
```

然后访问：

```text
http://127.0.0.1:8024/
```
