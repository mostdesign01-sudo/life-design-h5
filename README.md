# 人生设计蓝图 H5

一个单文件静态 H5，用斯坦福人生设计课的对话节奏，帮助用户完成：

- 四个仪表盘：健康、工作、娱乐、爱
- 真问题雷达：区分重力问题和可设计问题
- 工作观与人生观指南针
- 心流与能量地图
- 三个五年奥德赛计划
- 可复制、可下载的个人人生设计蓝图

## 方法背景

这个项目受到 Bill Burnett 和 Dave Evans 合著的 *Designing Your Life: How to Build a Well-Lived, Joyful Life* 启发。它在中文语境里常被称作《斯坦福大学人生设计课》。

这本书把设计思维带入人生选择：人生不是一道需要算出唯一标准答案的工程题，而更像一个可以反复试做、快速反馈、持续迭代的设计项目。它强调先看见自己当下的位置，再重新定义问题，分清哪些是无法直接改变的“重力问题”，哪些是可以通过行动、访谈和原型来验证的真问题。

本 H5 主要借鉴并产品化了这些方法线索：

- 健康、工作、娱乐、爱的四个仪表盘
- 重力问题与可设计问题的区分
- 工作观与人生观的指南针
- 心流、回血和消耗的能量地图
- 三个平行的五年奥德赛计划
- 低成本人生设计采访和原型行动

## 两位作者

**Bill Burnett** 是斯坦福设计教育者、设计思维实践者，也是 Life Design 课程体系的重要推动者。他的工作长期连接产品设计、设计教育和人生设计。

**Dave Evans** 是斯坦福设计教育者、创业者和组织实践者。他和 Bill Burnett 一起把设计思维从产品与服务设计延展到个人选择、职业路径和生活原型。

这不是一个官方授权产品，也不替代心理咨询、职业咨询或医疗建议。它只是一个个人学习和实践项目，用更轻、更可操作的界面，把 Life Design 的对话节奏带到浏览器里。

## 记录与隐私

H5 会把填写进度保存在当前浏览器本机的 `localStorage` 中，用来支持刷新后继续填写。它不会把你的回答上传到服务器、GitHub 或任何远端。点击页面右上角的“重新开始”按钮可以清空本机记录。

## AI 深度蓝图

项目已预留 AI 接入骨架：

- 前端：报告页里的“AI 接口”和“AI 深度蓝图”按钮
- 后端：[`server/`](server/) 中的 Node.js 最小 API 服务
- 提示词：[`prompts/life-design-system-prompt.md`](prompts/life-design-system-prompt.md)
- 方案文档：[`docs/AI_BACKEND_PLAN.md`](docs/AI_BACKEND_PLAN.md)

第一版推荐只在用户完成问答后调用一次 AI，生成更深入的真问题重定义、能量地图、奥德赛计划重写和 90 天原型计划。API Key 只放在后端服务器环境变量中，不要写进前端。

当前 AI 骨架还包含一个最小商业化机制：

- 每个浏览器每天免费 1 次 AI
- 超出后需要激活码
- 激活码由作者手动发放
- 后端记录调用次数、tokens、估算成本和成功/失败状态

## 本地使用

直接用浏览器打开 `index.html` 即可。

如果想用本地服务预览：

```bash
python3 -m http.server 8024
```

然后访问：

```text
http://127.0.0.1:8024/
```
